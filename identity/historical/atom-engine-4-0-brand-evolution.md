# Atom Engine 4.0 — Brand Evolution (Histórico)

**Versão:** 1.0
**Data:** 2026-05-13
**Status:** Historical (referência)
**Origem:** consolidação do Project "Atom Engine 4.0" no Claude.ai
**Período coberto:** Fev–Mar 2026

> Captura a evolução de identidade visual do ecossistema Atom (codename interno: MindRoot) durante o Project Claude "Atom Engine 4.0". Documento histórico — para a identidade canônica atual, ver `law/system_spec_atom-identidade_v1-3.md` e `identity/` (atual).

---

## Trajetória

Brand do ecossistema evoluiu em camadas sucessivas:

1. **MindMate logos** (3 conceitos exploratórios)
2. **Renomeação MindMate → MindRoot** com paleta consolidada
3. **MindRoot tree** (símbolo central)
4. **Brand Manual interativo Nível 1** (pentagrama 5 marcas)
5. **Triângulo satélite** (3 empresas órbita externa)
6. **Brand Manual Nível 2** (dashboard fractal por marca)

Esta sequência aconteceu antes da consolidação canônica em D-004 (Atom = nome público único). Vários conceitos foram absorvidos pela identidade v1.3.

---

## Camadas

### 1. MindMate logos

Três conceitos explorados como `mindmate-logos.html`:

- **Raízes** — três raízes (mente/coração/alma) saindo do mesmo ponto. Marcado como "favorito".
- **Semente** — semente com veias internas representando os três caminhos nutricionais, com broto emergindo no topo.
- **Anéis** — corte transversal de árvore com anéis concêntricos.

Wordmarks em Cormorant Garamond (Primary, Spaced Caps, Duo Tone, Italic, Sans Lowercase). Previews dark/light.

### 2. Renomeação MindMate → MindRoot

Iteração final como `mindroot-logos.html` com paleta consolidada:

- Golden `#c4a882`
- Terracotta `#d4856a`
- Green `#8a9e7a`

Wordmarks em Cormorant Garamond 300 com letter-spacing 4–14. Previews em contexto de app dark/light.

### 3. MindRoot tree (`mindroot-tree.html`)

Árvore como símbolo central:

- Galhos = módulos (6 módulos: `#mod_purpose/work/family/body/mind/soul`)
- Tronco com anéis de crescimento = tempo/ciclos
- Raízes = tríade Mind/Heart/Soul

Codificações subliminais: família nuclear (4 grãos no tronco = R-F-B-B = Rick, Fernanda, Bella, Ben) e número 6 (1+5 = Rick).

### 4. Brand Manual interativo Nível 1 (`brand-manual-interactive.html`)

Pentagrama com 5 marcas em vértices coordenados (P1 200,40 / P2 366,161 / P3 303,357 / P4 97,357 / P5 34,161):

- **P1 MindRoot** — Consciência (árvore)
- **P2 Atom Engine** — Energia (três órbitas ao redor de núcleo)
- **P3 AtlasLab** — Exploração (steel frame com crescimento orgânico)
- **P4 Yugar Commons** — Terra
- **P5 Muda** — Movimento

Cada marca com tabs: Visão Geral, Logos, Wordmarks, Geometria, Subliminais.

Escala universal `1/φ = 0.618034` (camada sagrada idêntica para todas as marcas). Vagalume percorrendo a estrela em 16s com rastro acumulativo. Zoom de entrada `scale(2.8→1)` em 2.2s. ~1150 linhas single-file.

### 5. Triângulo satélite (`triangle-satellite.html`)

Sistema geométrico próprio para as 3 empresas em órbita externa ao pentagrama:

- **Atlas Frames** — construção civil ativa em QLD, ABN 76 644 837 516
- **Atlas Modular Systems** — extensão modular, pré-operacional
- **Yugar Holdings** — governança/proteção

Geometria sagrada: triângulo áureo 72°-72°-36° derivado de φ. Paleta steel `#9a8e7e`. Versão V4 final em estilo isométrico, prisma triangular, paleta azul Atlas `#013475`, DM Sans + JetBrains Mono.

Esclarecimento importante: **satélites NÃO são vértices do pentagrama** — são corpos em órbita. Yugar Commons (P4) é entidade SEPARADA do triângulo (Yugar Holdings).

### 6. Brand Manual Nível 2 — Dashboard operacional fractal

Cada marca, ao ser aberta, revela 5 dimensões num pentagrama interno:

- **V1 Visão** — casca
- **V2 Modelo** — segundo anel
- **V3 Operação** — anel central
- **V4 Finanças** — anel profundo
- **V5 Jurídico** — cerne

Sistema "anéis de crescimento": uma cor base da marca em 5 profundidades lineares com piso de luminosidade 52% (dark) e shift de saturação +15/matiz +5°. Vagalume Nível 2 espirala pra dentro (3.5 voltas, 24s).

MindRoot foi o caso populado completamente — propósito, freemium A$9.99/mês, stack, break-even ~200 subs Pro.

## Design canon dessa fase

### Tipografia
- **Cormorant Garamond 300** — títulos serif
- **Inter** — body sans
- **JetBrains Mono** — labels/code
- Emoções sempre em palavras, nunca emoji

### Paleta MindRoot
| Token | Hex |
|---|---|
| bg | `#111318` |
| surface | `#1a1d24` |
| border | `#2a2d34` |
| text light | `#e8e0d4` |
| mind (golden) | `#c4a882` |
| heart (terracotta) | `#d4856a` |
| soul (green) | `#8a9e7a` |
| muted | `#a89478` |

Períodos do dia: aurora `#f0c674`, zênite `#e8e0d4`, crepúsculo `#8a6e5a`.

### Estética
- Dark theme only (à época)
- Border-radius 4px para botões primários
- Animações cubic-bezier(.16,1,.3,1)
- Tons subliminais codificados (datas familiares, numerologia) — sempre indiretos, nunca óbvios

## Como isso se relaciona com a identidade atual

D-004 (08 Mai 2026) consolidou: **Atom = nome público único do produto**. MindRoot rebaixado a codename interno. Vertex dissolvido como app separado.

Consequências dessa evolução de brand:

- **MindRoot tree** — conceito preservado como codename interno do app
- **Pentagrama 5 marcas** — descontinuado como estrutura de produto; vive como **preset pessoal "Pentagono Rick"** (ver Identidade v1.3 §2.6)
- **Triângulo satélite** — Atlas Frames continua como empresa; Atlas Modular evoluiu pra Atom (produto unificado); Yugar Holdings preservado
- **Paleta MindRoot** — superseded pela paleta Atom OS (background `#18171a`, accent `#7F77DD`, DM Sans 400/500)
- **Brand Manual Nível 2** — método (V1 Visão / V2 Modelo / V3 Operação / V4 Finanças / V5 Jurídico) absorvido como template de análise de venture
