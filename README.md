# Atom Engine Core

Universal data protocol for personal operating systems.

**Version:** Genesis v4.2 + AtomItem Schema v2
**Status:** Private — validation in progress
**Principle:** The schema is the contract. All systems are consumers.

## What's here

- `genesis/` — The universal data contract (schema, state machine, automations, SQL)
- `marco-zero/` — Operational guide (rituals, cockpit, Claude rules)
- `templates/` — AtomItem templates by type
- `sql/` — Supabase migration (Schema v2)
- `specs/` — Vision specs (Atom OS, future)
